How to use

  1. Open debug console
  2. Type "eulerMagic(<x>, <y>, <step>, <target>)", where (x, y) is the coordinate of the starting point, step is the step size x increments at, and target is the final value of x.
  3. To define a new differential equation, type "function dyOVERdx(x, y){return < expression>}", where expression is the differential equation in javascript syntax.
